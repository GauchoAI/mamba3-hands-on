# Findings

The operator curriculum scripts should be read as scaffolding. They establish
interfaces, artifacts, and a one-minute discipline, but they mostly rely on
features or harness logic that already contain the state needed by the policy.

That is acceptable as an introduction. It is not acceptable as the core claim.
The next experiment raises the bar by removing explicit stack-state features
from the learner input.
